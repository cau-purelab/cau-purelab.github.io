
# SVIL | Security Visual Intelligence Lab

Security Visual Intelligence Lab (SVIL)의 공식 홈페이지입니다. 
중앙대학교 노승민 교수님 지도하에 AI 보안, 신뢰할 수 있는 AI(Trustworthy AI), 그리고 Machine Unlearning 분야를 연구합니다.

## 🚀 연구 분야
- **Machine Unlearning**: 생성 모델(Diffusion, LLM)의 데이터 삭제 기술 연구
- **Secure Computer Vision**: 적대적 공격 방어 및 신뢰 가능한 시각 지능 시스템 구축
- **Trustworthy AI**: 인공지능의 안전성 및 프라이버시 보호 기술

## 🛠 Tech Stack
- **Framework**: React 19
- **Styling**: Tailwind CSS
- **Routing**: React Router (HashRouter)
- **AI Assistant**: Google Gemini API (@google/genai)
- **Icons**: Lucide (SVG), Lucide React
- **Typography**: Playfair Display, Inter, Noto Sans KR

## 📂 Project Structure
- `/pages`: 주요 페이지 (Home, Research, People, Publications)
- `/components`: UI 컴포넌트 (Navbar, LabAssistant)
- `/assets`: 프로필 사진 및 이미지 리소스 (input_file_0.png, input_file_1.png, input_file_2.png 등 저장 위치)
- `constants.tsx`: 연구실 데이터 통합 관리

## 📝 설치 및 실행 방법
1. 저장소를 클론합니다.
   ```bash
   git clone https://github.com/cheonbung/SVIL-Homepage.git
   ```
2. 의존성 패키지를 설치합니다.
   ```bash
   npm install
   ```
3. 로컬 개발 서버를 실행합니다.
   ```bash
   npm start
   ```
